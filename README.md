# BeatBot v2
BeatBot v2 is a Fortnite lobby bot implemented as a web application. It allows you to manage your Fortnite parties, matchmaking, and more directly through an easy-to-use web interface, without the need for Discord bots or third-party applications.

## Features

- **Party Management**: Create, join, and manage Fortnite parties.
- **Matchmaking**: Handle matchmaking tickets and queue for games.
- **Friends and Party**: Manage friend requests, set emotes and cosmetics.
- **Cross-Platform**: Works on phones, consoles, and desktops.
- **Secure**: Uses your own Epic Games account credentials.

## Benefits

- **Always Available**: Access your bot anytime through the web.
- **Easy to Use Interface**: Simple web UI, no complex setups.
- **No Discord Bot Bullshit**: Direct integration without relying on Discord servers.
- **Use Your Own Account**: Securely log in with your Epic Games credentials.
- **Less Risky**: No need to share accounts or use untrusted third-party tools.
- **Free**: Open-source and free to use.

## Access

BeatBot v2 is available online at [https://beat-yt.github.io/beatbot-v2/](https://beat-yt.github.io/beatbot-v2/)

## Usage

1. Log in with your Epic Games account.
2. Access the dashboard to manage your bot.
3. Invite the bot to your party.
